# Project1--Agriculture-Smart-Farming---Computer-Vision-for-Crop-Disease-Detection

An AI-powered smart agriculture project that uses Computer Vision and Deep Learning to detect crop diseases from leaf images. The model helps farmers identify plant diseases early, enabling timely treatment and improved crop yield.

## 🚀 Features
- Crop disease detection from leaf images
- Deep learning-based image classification
- Image preprocessing and augmentation
- Model training and evaluation
- Disease prediction on new images

## 🛠️ Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib
- Jupyter Notebook

## 📂 Project Structure
```
Project/
├── Agriculture & Smart Farming - Computer Vision for Crop Disease Detection.ipynb
├── dataset/
├── README.md
```

## ▶️ How to Run
1. Clone this repository.
2. Install the required Python libraries.
3. Open the Jupyter Notebook.
4. Run all cells to train and test the model.

## 🎯 Objective
To develop an intelligent system that automatically detects crop diseases from plant leaf images using deep learning techniques, helping farmers improve crop health and productivity.
